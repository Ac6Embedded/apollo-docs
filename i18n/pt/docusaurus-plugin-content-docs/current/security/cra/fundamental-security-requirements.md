## Visão geral

O Anexo I do CRA define os requisitos essenciais de cibersegurança para produtos com elementos digitais. Para equipas de sistemas embarcados é útil agrupá‑los em temas: **design seguro**, **atualizações seguras**, **gestão de vulnerabilidades**, **documentação** e **suporte**.[1]
