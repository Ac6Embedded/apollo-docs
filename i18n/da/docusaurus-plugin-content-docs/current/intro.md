# Apollo – overblik

Velkommen til **Apollo**, AC6’s læringsplatform med fokus på indlejret sikkerhed i forbundne og sikkerhedskritiske systemer.

Dokumentationen er organiseret omkring ét hovedspor:

- **Security** – begreber, lovgivning (herunder EU Cyber Resilience Act) og praktiske kontroller for indlejrede produkter.

Brug sidepanelet eller topmenuen til at navigere mellem sikkerhedsemnerne. Hver side er bevidst kort, praktisk og let at genbruge i designreviews, træning og audits.

## Sådan navigerer du

- Start med **Security → Overview** for at forstå, hvorfor indlejret sikkerhed og CRA er vigtigt.  
- Brug CRA‑undersektionen under Security til alt, der vedrører konformitet, dokumentation og processer.

Du kan skifte mellem **English**, **Français**, **Deutsch**, **Español**, **Português** og nordiske sprog via sprogmenuen i navbaren. Alle sprog deler den samme struktur, så links kan deles på tværs af teams.
