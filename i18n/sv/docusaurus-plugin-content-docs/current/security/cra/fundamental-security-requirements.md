## Översikt

Bilaga I i CRA definierar de väsentliga cybersäkerhetskraven för produkter med digitala element. För embedded‑team är det praktiskt att gruppera dem i teman: **secure design**, **secure updates**, **vulnerability handling**, **dokumentation** och **support**.[1]
