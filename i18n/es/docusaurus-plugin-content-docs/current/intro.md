# Apollo – vista general

Bienvenido a **Apollo**, la plataforma de aprendizaje de AC6 centrada en la seguridad embebida para sistemas conectados y críticos para la seguridad.

La documentación se organiza en un único recorrido principal:

- **Security** – conceptos, normativa (incluido el Reglamento europeo Cyber Resilience Act) y controles prácticos para productos embebidos.

Utiliza la barra lateral izquierda o la navegación superior para explorar los temas de seguridad. Cada página está pensada para ser breve, práctica y fácil de reutilizar en revisiones de diseño, formaciones o auditorías.

## Cómo navegar

- Empieza por **Security → Overview** para entender por qué la seguridad embebida y el CRA son importantes.  
- Utiliza la subsección CRA dentro de Security para todo lo relacionado con conformidad, documentación y procesos.

Puedes cambiar entre **English**, **Français**, **Deutsch**, **Español** y otros idiomas mediante el selector de idioma en la barra de navegación. Todas las lenguas comparten la misma estructura, de modo que puedas compartir enlaces con equipos mixtos.
