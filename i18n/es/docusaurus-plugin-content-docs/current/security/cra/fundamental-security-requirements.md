## Resumen

El Anexo I del CRA define los requisitos esenciales de ciberseguridad para los productos con elementos digitales. Para equipos embebidos es útil agruparlos en temas: **diseño seguro**, **actualización segura**, **gestión de vulnerabilidades**, **documentación** y **soporte**.[1]
