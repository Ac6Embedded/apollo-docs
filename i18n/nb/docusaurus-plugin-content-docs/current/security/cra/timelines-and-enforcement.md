## Tidslinjer

- Overgangsperioder og ikrafttredelser av CRA artikler og vedlegg.
- Produktkategoriers frister for samsvar.
- Varsler om end-of-support og plikter for oppdateringer.

## Håndheving

- Forpliktelser for produsenter, importører, distributører og leverandører.
- Håndheving via markedstilsyn, rapportering og sanksjoner.
- Plikt til tidlig varsling ved betydelige hendelser.

## Praktiske råd

- Planlegg releasekalendere i tråd med støtteperioder.
- Etabler sporbarhet fra krav til evidens for hver release.
- Koordiner med leverandørkjede om oppdateringsdistribusjon.
