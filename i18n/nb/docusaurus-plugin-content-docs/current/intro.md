# Apollo – oversikt

Velkommen til **Apollo**, AC6 sin læringsplattform med fokus på innebygd sikkerhet i tilkoblede og sikkerhetskritiske systemer.

Dokumentasjonen er organisert rundt ett hovedspor:

- **Security** – konsepter, regelverk (inkludert EUs Cyber Resilience Act) og praktiske kontroller for innebygde produkter.

Bruk sidepanelet eller toppmenyen for å navigere mellom sikkerhetstemaene. Hver side er bevisst kort, praktisk og lett å gjenbruke i designgjennomganger, opplæring og revisjoner.

## Slik navigerer du

- Start med **Security → Overview** for å forstå hvorfor innebygd sikkerhet og CRA er viktig.  
- Bruk CRA‑underseksjonen under Security for alt som gjelder konformitet, dokumentasjon og prosesser.

Du kan bytte mellom **English**, **Français**, **Deutsch**, **Español**, **Português** og nordiske språk via språkvelgeren i navbaren. Alle språk deler samme struktur, slik at lenker kan deles på tvers av team.
